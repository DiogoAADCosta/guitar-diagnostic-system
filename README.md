# Python Estudos

Este repositório funciona como um diário técnico do meu processo de aprendizado em Python.

A proposta não é reunir exercícios aleatórios, mas registrar de forma estruturada:
- como estou pensando os problemas;
- quais decisões tomo ao resolver cada exercício;
- quais dúvidas surgem no processo;
- e como os códigos, desafios e soluções vão se tornando mais complexos e elaborados com o tempo.

Os códigos aqui priorizam clareza de raciocínio e registro de aprendizado, não otimização ou performance profissional.

Organização do repositório:
- Cada pasta representa um conceito estudado (listas, loops, strings, etc.).
- Os arquivos contêm comentários explicando a lógica utilizada.
- Quando relevante, registro alternativas ou observações sobre como eu resolveria hoje.

Objetivo: construir uma base sólida de lógica e pensamento estruturado.
