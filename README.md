---

## Main Project

### 🎸 Guitar Diagnostic System

A diagnostic system designed to evaluate guitar skills  
through structured tests and rule-based analysis.

---

## How the System Works

1. The user selects a goal (e.g., playing songs, improvising)  
2. Relevant skill pillars are activated  
3. The system determines a starting level  
4. The user completes one or more tests:
   - Behavioral (pattern-based, no correct answers)
   - Validation (knowledge-based, with correct answers)
5. Each test generates structured data  
6. Results are combined into a final diagnostic  

---

## Design Focus

- Behavioral modeling  
- Structured data collection  
- Rule-based decision systems  
- Separation between validation and interpretation  
- Scalable test architecture  

---

## Engines

The system is built around two core engines:

- Behavioral Engine → interprets response patterns  
- Validation Engine → evaluates correctness and controls test flow
